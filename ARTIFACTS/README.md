# LexX Axiom v1.4 Artifacts

This directory contains the complete, production-ready Flutter app for LexX Axiom v1.4.

## Contents

- `pubspec.yaml`: Project configuration and dependencies.
- `lib/`: Dart source code.
- `assets/`: Shaders and themes.
- `config/`: Schema configurations.

## Running the App

1. Install Flutter 3.24+.
2. Run `flutter pub get`.
3. Run `flutter pub run build_runner build`.
4. Run `flutter run` for development or build for platforms.

## Features

- Transcendent white-pink holographic UI.
- Offline deterministic Lex Library.
- Chat, voice, file upload, DeFi validation, audit.
