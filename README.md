## Documentation

### Installation

```bash
# Clone repository
git clone https://github.com/AXI0MH1VE/lexx-axiom-app.git
cd lexx-axiom-app

# Setup Flutter environment
flutter pub get
flutter pub run build_runner build
```

### Usage

**UDP Verification**:
```bash
# Test UDP integrity
make ci

# Run artifacts (smoke test)
make smoke

# Recompute attestation
make attn
```

**Flutter Development**:
```bash
# Run development server
flutter run

# Build for production
flutter build apk  # Android
flutter build ios  # iOS
flutter build web  # Web
```
